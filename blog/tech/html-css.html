What is the difference between Margin, Border, and Padding?

What is Margin?

Margin is the space around an element's border. This space can be applied to the top, right, left, and/or bottom margins of an element.

What is Border?

Border is the outline of an element, and if no border is defined then it defaults to being "none". CSS allows the style, width, and colour of a border. For example, dotted, dashed, solid etc.

What is Padding?

Padding is the space between the content and border. This space can be applied to the top, right, left, and/or bottom padding of an element.

Margin + Padding Tip

When using CSS, margin and padding application can be simplified. If a property has four values, eg. padding: 20px 15px 25px 10px;, the order of those values goes top, right, bottom, left.

If a property has three values, eg. padding: 20px 15px 25px;, the order of those values goes top, right and left, bottom.

If a property has two values, eg. padding: 20px 15px;, the order of those values goes top and bottom, right and left.
