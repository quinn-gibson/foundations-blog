Control Flow and Loops

Control flow is the order in which statements of code are executed. Loops are blocks of code that will be executed until a condition is met. We can understand this process as if we were emptying the dishwasher. The dishwasher would get opened, and then a loop of taking dishes out and putting them away would be done until the dishwasher is empty.

Document Object Model

The document object model, or the DOM, is a tree-like representation of a webpage's structure using objects. The DOM is created by the browser when a webpage is loaded, and is used by JavaScript to access and change elements of the webpage. You can access elements by using 'getElementById()' or 'getElementsbyClassName()'. ID returns a single element with that ID and class name returns an array of elements with that class name.
